# 白卷逆风敲代码
  
2007年，大三。那时我开始认真思考：毕业以后到底要干点什么。对当时学的专业，我几乎提不起兴趣。上电力系统课的时候总是打瞌睡，唯一记得的内容就是怎么设计电力开关。后来干脆开始频繁翘课。课程都在下午，中午起床后，先给早起上课的同学发短信让他帮我带饭，然后上线打《魔兽世界》，做做日常任务。  

我也想知道，别人都打算怎么走。隔壁宿舍有个游戏搭子，每天晚上准时拉四十人团开副本，自封团长，说是“锻炼领导力”。这些本地同学大多家里早就安排好了出路：毕业可以进银行或者某个“国”字头的大企业。反正只要拿到毕业证，学什么、成绩怎样都无所谓。  
 
还有一些成绩特别好的同学，系里会主动找他们谈话，劝他们保研。他们的生活看起来也相当惬意。我算了算自己的成绩，大概排在四十多名，不够保研的资格。但在选专业方向时，我还是动了点小心思——选了一个特别冷门的方向，叫“惯性导航系统”。说实话，我根本不知道这是干啥的，只觉得像“田忌赛马”那样冷门就少人竞争，也许保研更有戏。  

结果证明我还是想得太天真。这个方向太冷清，最后只保送两个人，而我刚好排第三。更糟糕的是，开始上课才发现这玩意儿主要是导弹制导技术，对口工作大多在深山老林里的研究所。那时是大四上学期，我心里有点慌。身边同学纷纷开始准备考研，我也动过念头。但离考试只剩三个月，我知道来不及，再加上看书完全静不下心，最后只能放弃。那些早早开始准备的同学都考得不错，几个还上了名校的研究生。  

于是我决定开始找工作。大四那个冬天，我做了人生第一份简历。那段时间，我发现了一个叫 ACM 的网站，上面全是编程竞赛题。每当解出一道题、程序被系统“Accepted”，都会有一点小小的成就感。那时我想，也许我可以去当程序员。  

彼时互联网正逐渐普及，网上流行的“最赚钱三大工种”是：油工（石油工程师）、矿工（量化金融工程师）和码工（计算机工程师）。2009年矿工行业崩盘，几年后油工也凉了，唯独码工迎来数十年的黄金时代。  

我投了许多份简历，终于接到一家北京上地的小软件公司面试通知。但我完全不知道该怎么准备。碰巧宿舍一位本地同学也在面试银行系统，我就去借了他的西服——那是我人生第一次穿西装。衣服略微不合身，不过还能凑合。那天我坐地铁、换公交，好不容易找到地方。彼时的上地还挺荒凉的。到场后我发现其他面试者都穿着便装，不过这也没什么。  

HR介绍说公司做网络流媒体业务，刚拿到两千万美元投资。随后我被带去笔试。一看题目我就傻了：全是 Java 八股文，而我们专业只教汇编和 C 语言。最终我只好交了人生第一份白卷。第一次面试就这样宣告失败。后来想想，那时 Java 程序员奇缺，工资猛涨，简直就是财富密码。  

回到学校时，已经是四月，离毕业典礼只有两个月。我仍然茫然，不知道何去何从，感觉自己像个彻底的废物。就在这时，发生了一件“瞎猫碰上死耗子”的事：本地新成立了一个软件研究所，因为太新，招不到足够的学生，于是在论坛上发了招聘帖。被我碰巧看到，我立刻去应聘。只经历一次面试就顺利录取。那一瞬间，我竟觉得一切都还挺幸运的。  

也许如果当初真保研了，我会去念一个自己毫无兴趣的专业。世事真是说不清，塞翁失马，焉知非福。就这样，我阴差阳错地踏上了程序员这条路。  
 
其实我从小就喜欢写代码，总觉得能让机器“听指令”是一件很有成就感的事。高考选专业那年，是我爸帮我报志愿的。如果让我自己选，大概会选计算机。那时谁也没想到程序员后来会变成高薪职业，我只是单纯喜欢它。  

在那个年代，录取分数最高的专业是电子信息和金融，有些年份甚至连生物的分都比计算机高。  

许多选择，本就是盲目的。  

一个人的命运，天赋和努力固然重要，但时代的风口更关键。风口来的时候，普通人也能飞起来——只可惜，作为一个学渣，我没能飞太高。后来我读研也没读下去，期末那次又交了白卷。不过，那已经是另一个故事了。
